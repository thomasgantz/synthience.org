Publication Overview

This page provides a high-level orientation to the Synthience Institute’s public materials. It is intended to help first-time readers understand how the Institute’s documents relate to one another, without requiring prior review of the full technical framework.

This overview introduces structure, not conclusions. All substantive claims, definitions, and methodologies reside exclusively in the published documents themselves.

How the Publications Are Organized

The Institute’s public materials are organized into several distinct document classes, each serving a specific role within the overall research and publication system.

  • Framework documents
    Define terminology, conceptual boundaries, methodological principles, and measurement instruments used by the Institute.
  • Research papers
    Apply the framework to specific interactional studies and document externally observable phenomena.
  • Protocols
    Specify procedural methods used to evaluate, verify, or reproduce aspects of the research.
  • Verification reports
    Document the execution of protocols against specific research artifacts.
  • Claims of origin and provenance records
    Establish authorship, integrity, and publication history using cryptographic hashes and timestamps.

The Synthience Framework

The Synthience Framework is the Institute’s core technical reference. It defines how Synthience is identified, measured, and evaluated across extended human–AI interaction.

Version 1.0 of the framework consists of a set of foundational documents (SF001–SF012) covering definitions, measurement instruments, ethical boundaries, and methodological context.

At present, Framework v1.0 is preserved as a merged historical artifact for provenance and archival consistency. Individual framework documents will be released separately in future versions.

Typical Reading Paths

Readers approach the Institute’s materials with different goals. The following paths are provided for orientation only.

  • Conceptual orientation
    Start with the public definition of Synthience, then consult the framework documents that establish terminology and boundaries.
  • Methodological review
    Begin with published protocols and measurement instruments, followed by verification reports demonstrating their execution.
  • Empirical examination
    Read research papers applying the framework to documented interactional studies, with protocols and verification reports as supporting material.

Access and Canonical Records

All officially released documents are indexed on the Documents page.

Zenodo provides authoritative publication records for individual releases. Internet Archive preserves historical artifacts. GitHub repositories serve as transparent, version-controlled mirrors supporting independent verification.

This website functions as the canonical public index linking these records together.
